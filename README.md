# EndrollMate

## Requires

* node packages
  - grunt-cli
  - typescript@next (recommended)
  - babel

## How to build

1. $ cd ts/jquery.keyframes.d.ts
2. $ git submodule init
3. $ git submodule update
4. $ grunt
